## 📦 Deployment

### GitHub Pages (Recommended)

1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to Settings > Pages
4. Select "main" branch as source
5. Your site will be live at: `https://yourusername.github.io/repository-name`

### Custom Domain (Optional)

To use a custom domain like `akashananthula.com`:
1. Purchase a domain from providers like Namecheap, GoDaddy, or Google Domains
2. Add a CNAME file with your domain name
3. Configure DNS settings in your domain provider
4. Update GitHub Pages settings with your custom domain
