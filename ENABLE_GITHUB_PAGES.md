# 🚀 Final Step: Enable GitHub Pages

Your code is now on GitHub! Just one more step to make it live.

## ⚡ Quick Steps (Takes 1 minute):

1. **Go to your repository:**
   👉 [https://github.com/Akaash-45/Blog](https://github.com/Akaash-45/Blog)

2. **Click the "Settings" tab** (top right of the page)

3. **Click "Pages"** in the left sidebar

4. **Under "Source":**
   - Select "Deploy from a branch"
   - Branch: Select "main"
   - Folder: Select "/ (root)"
   - Click "Save"

5. **Wait 2-3 minutes** and refresh the page

6. **You'll see a green box saying:**
   ```
   ✅ Your site is live at https://akaash-45.github.io/Blog/
   ```
