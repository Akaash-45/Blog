# 📄 Resume Link Examples

## How to Add Your Blog Link to Resume

### Option 1: Header Section (Recommended)
```
AKASH ANANTHULA
Email: akash@example.com | Phone: +1-XXX-XXX-XXXX
Blog: https://yourusername.github.io/akash-blog | LinkedIn: linkedin.com/in/akashananthula
GitHub: github.com/akashananthula
```

### Option 2: With Your Name as Hyperlink (Digital Resume)
```
AKASH ANANTHULA
(Make "AKASH ANANTHULA" a clickable link to your blog)

Contact: akash@example.com | +1-XXX-XXX-XXXX
LinkedIn: linkedin.com/in/akashananthula | GitHub: github.com/akashananthula
```

---

## 🎯 Best Practices

1. **Keep it short**: Use shortened URLs if needed (bit.ly, tinyurl)
2. **Make it clickable**: For digital resumes (PDF with links)
3. **Test the link**: Before sending your resume
4. **Use HTTPS**: Always use secure links
5. **Professional domain**: Consider custom domain for extra professionalism

---

## 🔗 URL Shortening (Optional)

If your GitHub Pages URL is too long, you can:

1. **Use a custom domain** (Best option)
   - `akashananthula.com` instead of `yourusername.github.io/akash-blog`

2. **Use URL shorteners** (Quick fix)
   - bit.ly: Create custom short link like `bit.ly/akash-blog`
   - tinyurl.com: Create custom link
   - rebrandly.com: Professional branded links

## 🎨 For Different Resume Formats

### LaTeX Resume
```latex
\href{https://yourusername.github.io/akash-blog}{Akash Ananthula}
```

### Word Document
1. Type your name: AKASH ANANTHULA
2. Select the text
3. Right-click > Hyperlink
4. Paste your blog URL
5. Click OK

### Google Docs
1. Type your name: AKASH ANANTHULA
2. Select the text
3. Insert > Link
4. Paste your blog URL
5. Click Apply

### PDF Resume
Use tools like Adobe Acrobat to add clickable links to existing PDFs
